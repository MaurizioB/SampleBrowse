SampleBrowse
============

An audio sample browser and manager.

Requirements
------------

- Python >= 3.4.5
- PyQt5 >= 5.7.1
- pysoundfile >= 0.9.0

Install
-------

    `./setup.py install [--user] [--record files.txt]`

- `--user` to install locally instead of system-wide
- `--record` to keep track of installed files
